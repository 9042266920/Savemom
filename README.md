# SaveMom UI Project

React.js project implementing the Figma design for SaveMom Virtual Maternal Healthcare.

## Project Description
- Fully responsive UI.
- Dynamic components with reusable structure.
- Designed according to Figma design.
- Optional: data can be fetched from backend API.

## How to Run Locally
1. Clone the repository:<br>
Open your terminal and run:<br>
git clone https://github.com/9042266920/Savemom.git<br>
This will download the project to your local machine.

2. Navigate to the Project Folder:<br>
cd Savemom

3. Install all required Node.js packages using:<br>
npm install<br>
This will install React, Vite, and other necessary libraries.

4. Start the Development Server:<br>
npm start<br>
The app will open automatically in your default browser.<br>
If not, visit http://localhost:5173 (Vite default port).
